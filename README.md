# 📘 SQL Daily Challenges

This repository is a collection of daily SQL problems along with their solutions. The goal is to build strong SQL skills through consistent practice and real-world problem solving.

---

## 🚀 Objective

- Improve SQL query writing skills  
- Practice real interview-level questions  
- Strengthen problem-solving ability  
- Prepare for Data Analyst and Data Engineer roles  

---

---

## 🧠 Topics Covered

- SELECT statements  
- WHERE, GROUP BY, HAVING  
- Joins (INNER, LEFT, RIGHT)  
- Subqueries  
- Window functions  
- Aggregations  
- Data cleaning queries  
- Real-world business problems  

---

## 💡 How to Use

1. Pick a daily problem  
2. Try solving it on your own  
3. Compare with the provided solution  
4. Optimize your query if possible  

---

## 🎯 Target

Consistently solve SQL problems to become confident in:
- Writing optimized queries
